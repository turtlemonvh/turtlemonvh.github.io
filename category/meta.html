This first blog post explains how the blog was made.

  • set up site using instructions here: https://pages.github.com/
  • my site is: turtlemonvh.github.io: https://github.com/turtlemonvh/turtlemonvh.github.io
  • then I cloned the site down locally so I could work on it
  • I followed the instructions here to set up pelican: http://docs.getpelican.com/en/stable/quickstart.html
  • I ran pelican-quickstart. Here are the values I used

Where do you want to create your new web site? [.] What will be the title of this web site? Systems Doing Who will be the author of this web site? turtlemonvh What will be the default language of this web site? [en] Do you want to specify a URL prefix? e.g., http://example.com (Y/n) Y What is your URL prefix? (see above example; no trailing slash) http://turtlemonvh.github.io/ Do you want to enable article pagination? (Y/n) Y How many articles per page do you want? [10] What is your time zone? [Europe/Paris] America/New_York Do you want to generate a Fabfile/Makefile to automate generation and publishing? (Y/n) n Do you want an auto-reload & simpleHTTP script to assist with theme and site development? (Y/n) Y Done. Your new project is available at /Users/timothy/Projects/turtlemonvh.github.io

  • I cloned this repository into ~/pelican-themes: https://github.com/getpelican/pelican-themes

  • You can follow these directions if you want a custom domain name (like stuartsshop.com: https://help.github.com/articles/setting-up-a-custom-domain-with-github-pages/)
